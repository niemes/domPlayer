# DomPlayer 

## Description: 

DomPlayer is a simple videoPlayer with feature usefull for projection.
- Move / Resize the video in the view for a projection. 
- Dom mode - Dom deformation on dom / half dome.
- playlist support

## Install : 

Download the last release.
